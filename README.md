## 🇬🇧 English

Welcome to the **HP Cyberpunk Portfolio**, an ultra-modern, futuristic hacker-style personal portfolio website for Phan Van Hoang. The project was constructed using a cutting-edge tech stack to deliver a visually stunning, highly interactive, and performant user experience.

### 🌟 Features
* **Futuristic Hero Section**: Glitched text, matrix typing animations, and neon button glows.
* **Interactive Hacker Terminal**: A floating, draggable terminal where users can type commands (like `help`, `skills`, `projects`) to navigate the site.
* **AI Chatbot Assistant**: A built-in virtual assistant that responds to questions about the developer's background, technologies, and projects.
* **3D Globe Network**: A Three.js powered rotating globe visualizing global technology networks.
* **Skill Radar Chart**: Data visualization of technical proficiencies using Recharts.
* **Active Archives (Projects)**: Modern glassmorphism project cards with hover-triggered ambient glow effects.
* **Dynamic GitHub Integration**: Fetches real-time stats and repositories using the GitHub API.
* **System Logs Timeline**: Animated chronological timeline of education and professional experience.
* **Contact Protocol**: A highly stylized secure transmission form mimicking a hacker payload delivery system.
* **Cyberpunk UI Aesthetics**: Built with customized Tailwind CSS v4, featuring a deep dark mode, CRT scanlines, and a dynamic interactive particle background.

### 🛠️ Tech Stack
* **Frontend Framework**: React 18, TypeScript, Vite
* **Styling**: Tailwind CSS v4, Custom CSS Animations
* **Interactions & Animations**: Framer Motion
* **3D & Canvas**: Three.js, `@react-three/fiber`, `@react-three/drei`
* **Background Effects**: `@tsparticles/react`
* **Data Visualization**: Recharts
* **Icons**: Lucide React
* **API Calls**: Axios

### 💻 Installation & Setup

1. **Clone the repository** (if applicable) or navigate to the project directory:
   ```bash
   cd profile_hoangphan
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the application**:
   Open your browser and navigate to `http://localhost:5173`.

### 📂 Folder Structure
* `src/components/layout/`: Core wrappers (Navbar, Footer, Layout)
* `src/components/sections/`: Main content blocks (Hero, Skills, Projects, Experience, Contact, GitHub)
* `src/components/widgets/`: Floating interactive UI elements (Terminal, AI Chatbot)
* `src/components/ui/`: Reusable smaller components (GlitchText, 3D Globe, Particles)
* `src/index.css`: Global styles, CSS variables, and keyframe animations

## 🇻🇳 Tiếng Việt

Chào mừng đến với **HP Cyberpunk Portfolio**, một trang web hồ sơ cá nhân mang phong cách hacker tương lai cực kỳ hiện đại dành cho Phan Văn Hoàng. Dự án được xây dựng với các công nghệ web tiên tiến nhất nhằm mang lại trải nghiệm thị giác ấn tượng và tính tương tác cao.

### 🌟 Tính Năng Nổi Bật
* **Hero Section Tương Lai**: Hiệu ứng chữ bị lỗi (glitch), hiệu ứng gõ phím ma trận và nút bấm phát sáng neon.
* **Terminal Hacker Tương Tác**: Một cửa sổ terminal nổi có thể gõ lệnh (như `help`, `skills`, `projects`) để điều hướng trang web một cách độc đáo.
* **Trợ Lý Trí Tuệ Nhân Tạo (AI Chatbot)**: Chatbot ảo được tích hợp để trả lời các câu hỏi về kinh nghiệm, công nghệ và dự án của lập trình viên.
* **Bản Đồ Mạng Lưới 3D**: Quả địa cầu quay 3D được tạo bởi Three.js, hiển thị các nút mạng và đường kết nối phát sáng.
* **Biểu Đồ Radar Kỹ Năng**: Trực quan hóa dữ liệu về mức độ thành thạo các kỹ năng bằng thư viện Recharts.
* **Thẻ Dự Án Kính Hóa (Glassmorphism)**: Giao diện thẻ dự án hiện đại với hiệu ứng phát sáng mờ ảo khi di chuột.
* **Tích Hợp GitHub Động**: Tự động gọi API GitHub để hiển thị các dự án nổi bật, số lượng người theo dõi và trạng thái hoạt động.
* **Dòng Thời Gian (Timeline)**: Hiệu ứng cuộn hiển thị lịch sử học tập và làm việc.
* **Giao Diện Liên Hệ Bảo Mật**: Form liên hệ được thiết kế theo phong cách truyền tải dữ liệu mật mã của hacker.
* **Thẩm Mỹ Cyberpunk UI**: Sử dụng Tailwind CSS v4 tùy chỉnh, chế độ nền tối sâu, hiệu ứng màn hình CRT và phông nền hạt (particles) tương tác động.

### 🛠️ Công Nghệ Sử Dụng

* **Giao Diện (Styling)**: Tailwind CSS v4, Custom CSS Animations
* **Hiệu Ứng (Animations)**: Framer Motion
* **Đồ Họa 3D**: Three.js, `@react-three/fiber`, `@react-three/drei`
* **Hiệu Ứng Nền**: `@tsparticles/react`
* **Trực Quan Hóa Dữ Liệu**: Recharts
* **Biểu Tượng (Icons)**: Lucide React
* **Gọi API**: Axios
